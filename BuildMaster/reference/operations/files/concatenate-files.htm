Concatenate Files

Concatenates files on a server.

Script usage:

Concatenate-Files(
	File: <text>,
	[Directory: <text>],
	[Include: <@(text)>],
	[Exclude: <@(text)>],
	[Encoding: <text>],
	[Separator: <text>]
);

This operation may be prefixed with Files::, although this is a built-in namespace and isn't really necessary.
