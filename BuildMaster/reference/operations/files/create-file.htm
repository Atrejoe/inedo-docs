Create File

Creates a file on a server.

Script usage:

Create-File(
	Name: <text>,
	[Text: <text>],
	[Overwrite: <true/false>],
	[FileMode: <text>]
);

This operation may be prefixed with Files::, although this is a built-in namespace and isn't really necessary.
