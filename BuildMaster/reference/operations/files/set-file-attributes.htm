Set File Attributes

Sets or clears attributes on matching files.

Script usage:

Set-FileAttributes(
	[Directory: <text>],
	[Include: <@(text)>],
	[Exclude: <@(text)>],
	[ReadOnly: <true/false>],
	[Hidden: <true/false>],
	[System: <true/false>],
	[Verbose: <true/false>]
);

This operation may be prefixed with Files::, although this is a built-in namespace and isn't really necessary.
