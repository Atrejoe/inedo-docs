Ensure Application

Ensures the existence of an application within an IIS site.

Script usage:

IIS::Ensure-Application(
	Site: <text>,
	Path: <text>,
	[AppPool: <text>],
	[PhysicalPath: <text>],
	[LogonMethod: <integer>],
	[Credentials: <text>],
	[UserName: <text>],
	[Password: <text>],
	[Exists: <true/false>]
);

Example:

# ensures that the hdars application is present on the web server
IIS::Ensure-Application(
    Site: Hdars,
    Path: /hdars,
    PhysicalPath: C:\hdars
);
