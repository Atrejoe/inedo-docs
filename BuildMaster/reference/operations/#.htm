BuildMaster

  • Create Release Note - Attaches a release note to the current release.
  • Create Release Package - Creates a new release package in BuildMaster of an application, and optionally waits for it to complete.

  • Manual Operation - Halts the execution until an individual or group completes a specified task.
  • Set Release Package Number - Sets the release package number of the currently executing release package.
  • Set Release Variable - Creates or updates a variable on a release or package.
