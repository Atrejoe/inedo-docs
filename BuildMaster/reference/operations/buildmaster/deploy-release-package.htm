Deploy Release Package

Deploys a package in BuildMaster of an application, and optionally waits for it to complete.

Script usage:

Deploy-Package(
	Application: <text>,
	[ReleaseNumber: <text>],
	[From: <text>],
	[To: <text>],
	[Force: <true/false>],
	[Wait: <true/false>],
	[FailIfCannotDeploy: <true/false>]
);

This operation may be prefixed with Core::, although this is a built-in namespace and isn't really necessary.
