Create Release

Creates a release in another application.

Script usage:

Create-Release(
	Application: <text>,
	ReleaseNumber: <text>,
	Pipeline: <text>,
	[ReleaseName: <text>],
	[Variables: <%(key1: value1, ...)>]
);

This operation may be prefixed with Core::, although this is a built-in namespace and isn't really necessary.
