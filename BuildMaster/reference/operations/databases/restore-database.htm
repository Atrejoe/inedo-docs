Restore Database

Uses a Database Connection to restore a database from a file on disk.

Script usage:

Restore-Database(
	Connection: <text>,
	From: <text>,
	Name: <text>,
	[Verbose: <true/false>]
);

This operation may be prefixed with Core::, although this is a built-in namespace and isn't really necessary.
