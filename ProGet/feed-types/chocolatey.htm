A Chocolatey feed is actually a type of NuGet feed. Functionally, it behaves exactly the same as a NuGet feed, but the user interface is optimized more for Chocolatey packages.

Checkout the webinar below, presented by Chocolatey founder, Rob Reynolds. Rob reviews the benefits of ProGet and Chocolatey, and takes you through a demo of how to use ProGet and Chocolatey in conjunction, for a complete Windows package management solution.
