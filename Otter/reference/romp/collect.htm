The collect command runs a configuration plan and writes the collection results to the console.

Usage

romp collect <collectPlan.otter>

The collectPlan.otter argument specifies the file name of an OtterScript configuration plan.

Behavior

Romp will perform the collection against the local machine, display the results on the console, and exit.

Examples

Perform a collection using the plan contained in c:\plans\ensureWebsites.otter:

romp collect c:\plans\ensureWebsites.otter

Perform a collection using the plan contained in c:\plans\ensureWebsites.otter, and load templates from c:\templates:

romp --lib=c:\templates collect c:\plans\ensureWebsites.otter
