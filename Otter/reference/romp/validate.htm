The validate command parses a plan and reports any errors or warnings.

Usage

romp validate <plan.otter> [--simulate]

The plan.otter argument specifies the file name of an OtterScript plan.

Behavior

Romp will attempt to parse the specified plan file and display any errors or warnings to the console.

Examples

Validate the plan contained in c:\plans\createDevEnvironment.otter:

romp validate c:\plans\createDevEnvironment.otter
