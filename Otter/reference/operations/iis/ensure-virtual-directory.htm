Ensure Virtual Directory

Ensures the existence of a virtual directory within an IIS site.

Operation type:

Ensuring - this operation will ensure a specified configuration exists on a remote server after comparing its current state to the desired state.

Script usage:

IIS::Ensure-VirtualDirectory(
	Site: <text>,
	Path: <text>,
	[PhysicalPath: <text>],
	[LogonMethod: <integer>],
	[Credentials: <text>],
	[UserName: <text>],
	[Password: <text>],
	[Exists: <true/false>]
);

Example:

# ensures that the hdars virtual directory pool is present on the web server
IIS::Ensure-VirtualDirectory(
    Site: Hdars,
    Path: /hdars,
    PhysicalPath: C:\hdars
);
