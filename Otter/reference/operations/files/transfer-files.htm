Transfer Files

Copies files from a directory on a source server to a directory on a target server.

Operation type:

Executing - this operation will perform a specified action or actions on a remote server.

Script usage:

Transfer-Files(
	[Include: <@(text)>],
	[Exclude: <@(text)>],
	[FromDirectory: <text>],
	[FromServer: <text>],
	ToDirectory: <text>,
	[ToServer: <text>],
	[Verbose: <true/false>],
	[DeleteTarget: <true/false>]
);

This operation may be prefixed with Files::, although this is a built-in namespace and isn't really necessary.
