Push Package

Uploads a zip file containing the contents of a Universal Package to a ProGet feed.

Operation type:

Executing - this operation will perform a specified action or actions on a remote server.

Script usage:

ProGet::Push-Package(
	[Credentials: <text>],
	Feed: <text>,
	FilePath: <text>,
	[Group: <text>],
	[Name: <text>],
	[Version: <text>],
	[Description: <text>],
	[Title: <text>],
	[Icon: <text>],
	[Dependencies: <@(text)>],
	[Server: <text>],
	[UserName: <text>],
	[Password: <text>]
